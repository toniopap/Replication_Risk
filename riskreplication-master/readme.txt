This repository provides the whole code and data used in the paper

Open the file "main.R" and source it 
Some scripts are commented out because the take long to run. If you want to run them, uncomment them in the file "main.R"

To generate the appendix html file, knit the file "Appendix.Rmd"

The original data and code from BJR2014 is in the folder replicate_original.

The file "replication_ml.dta"" is a Stata dataset prepared for ML estimation in Stata.
